# Versakit UI

Versakit UI 是一个现代化的 React 组件库，专为桌面应用程序设计，提供了一套高度可定制的组件解决方案。

## 特点

- 🚀 &nbsp; 提供高质量的通用 React 组件
- 🎨 &nbsp; 使用 Tailwind CSS 实现高度可定制的样式
- 🛠️ &nbsp; 支持按需导入
- 🌍 &nbsp; 支持国际化
- 💡 &nbsp; TypeScript 支持
- 🌙 &nbsp; 暗黑模式支持
- ♿ &nbsp; 注重无障碍设计

## 安装

```bash
# 使用 npm
npm install @versakit/react

# 使用 yarn
yarn add @versakit/react

# 使用 pnpm
pnpm add @versakit/react
```

## 使用

```tsx
import React from "react";
import { Button, Card, Chip, Badge } from "@versakit/react";
import "@versakit/react/dist/main.css";

function App() {
  return (
    <div>
      <Card>
        <Card.Header>
          <Card.Title>示例卡片</Card.Title>
          <Card.Description>这是一个包含多个组件的示例</Card.Description>
        </Card.Header>
        <Card.Content>
          <div className="flex gap-2 mb-4">
            <Chip>标签1</Chip>
            <Chip variant="primary">标签2</Chip>
            <Chip variant="success" closable>
              可关闭标签
            </Chip>
          </div>
          <Button>
            消息 <Badge content="5" variant="error" position="top-right" />
          </Button>
        </Card.Content>
        <Card.Footer>
          <Button variant="primary">确认</Button>
          <Button variant="outline">取消</Button>
        </Card.Footer>
      </Card>
    </div>
  );
}
```
