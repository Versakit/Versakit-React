# Versakit UI

Versakit UI 是一个现代化的 React 组件库，专为桌面应用程序设计，提供了一套高度可定制的组件解决方案。

## 特点

- 🚀 &nbsp; 提供高质量的通用 React 组件
- 🎨 &nbsp; 使用 Tailwind CSS 实现高度可定制的样式
- 🛠️ &nbsp; 支持按需导入
- 🌍 &nbsp; 支持国际化
- 💡 &nbsp; TypeScript 支持
- 🌙 &nbsp; 暗黑模式支持
- ♿ &nbsp; 注重无障碍设计

## 安装

```bash
# 使用 npm
npm install @versakit/ui

# 使用 yarn
yarn add @versakit/ui

# 使用 pnpm
pnpm add @versakit/ui
```

## 使用

```tsx
import React from "react";
import { Button } from "@versakit/ui";
import "@versakit/ui/dist/style.css";

function App() {
  return (
    <div>
      <Button>点击我</Button>
    </div>
  );
}
```
